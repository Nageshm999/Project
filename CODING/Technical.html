We usually interchangeably use the term HTML elements and tags, but technically both are different. an HTML tag is just a character insude the angle bracket<>, whereas the HTML element is a collection of starting tag, its attribute, content and end tag, For example:

HTML Heading

To display the section heading, title or subtitle we can use the HTML heading tags. In HTML 5 we have 6 heading tags start from <h1> up to <h6>, where <h1> specify the largest heading and <h6>represnt the smallest or sub heading. if the content is specified vy heading tags, then it would be displayed large and bold as compared to other text content present on the web-page Example 

   
    <h1>First Heading </h1>
    <h2>Second Heading</h2>
    <h3>Third Heading</h3>
    <h4> Fourth Heading</h4>
    <h5>Fifth Heading</h5>
    <h6>Sixth Heading</h6>

HTML Formatting

In Html, we have many special elements that can provide special meaning to text content.

HTML Elements Decscription
<b> Bold the text
<Strong> An alternative for <b>, which tell that this text is essential.
Italic the text
<em> Similar to italic but used when we want emphasize the text.
<mark> It marks the text with a default "yellow" background color
<small> Decrease the text size
<del> It prints a cross line over the text.
<ins> Represnts the inserted text by putting an underline.
<sub> This element is used to display the subscript.
<sup> It can make a text superscript.
